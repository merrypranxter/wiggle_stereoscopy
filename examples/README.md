# Wiggle Examples

A few cute and creative demos to explore wiggle stereoscopy beyond the core particle scene. Serve the repo (e.g., `npx serve .`) and open these pages directly in your browser.

- **01-hearts.html** — Floating hearts in a pastel field. Adjust the frequency slider to change the wiggle speed. Depth layering from near to far hearts illustrates how the parallax varies with distance.
- **02-text.html** — A simple text wiggle. Use the frequency slider to change how fast the text shifts and the crossfade slider to blend between hard cuts and smooth sinusoidal motion.
- **03-starfield.html** — A 2D starfield with baseline and frequency controls. Stars near the viewer shift more than distant ones, creating a convincing depth effect without WebGL.

These examples are intentionally lightweight—no WebGL needed—so you can quickly test wiggle concepts, or copy the patterns into your own projects. For the full 3D particle implementation, see the root `index.html`.
